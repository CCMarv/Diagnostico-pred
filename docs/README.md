# Documentación del proyecto

Esta carpeta contiene toda la documentación del proyecto organizada según el Estándar de Documentación del equipo (Diátaxis):

- `tutoriales/` — tutoriales y quickstarts.
- `guias/` — guías prácticas y procedimientos.
- `referencia/` — referencia técnica y catálogos.
- `explicaciones/` — explicaciones, decisiones y material contextual.
- `adr/` — Architecture Decision Records numerados.
- `reporte/` — reportes académicos y artefactos finales.
- `implementacion_modelos/` — documentación centralizada del catálogo de modelos.

Lee `referencia/estandar-de-documentacion.md` para los principios y la arquitectura de documentación.
